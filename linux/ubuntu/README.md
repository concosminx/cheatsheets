# Install Docker

## preview script steps before running

```shell
curl -fsSL https://get.docker.com -o get-docker.sh
DRY_RUN=1 sudo sh ./get-docker.sh
```

## run the script
```shell
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
```

Check [documentation](https://docs.docker.com/engine/install/ubuntu/)
